# Examples

Practical code examples to help you build with SLM Builder.

## Example Categories

### [Getting Started](getting-started)
Basic usage patterns including data loading, training, and export.

### [Database Integration](database-integration)
Connect to PostgreSQL, MySQL, MongoDB, Redis, Elasticsearch, and more.

### [API Integration](api-integration)
Fetch training data from REST APIs, GraphQL, and webhooks.

### [Advanced Examples](advanced-examples)
Production-ready examples including multi-model comparison, distributed training, and deployment.

## Running Examples

All examples are self-contained and can be run independently:

```bash
# Install with examples dependencies
pip install slm-builder[all]

# Run an example
python examples/basic_qa.py
```

## Prerequisites

Most examples require:
- Python 3.8+
- SLM Builder installed
- Appropriate dependencies for specific features

See individual example pages for specific requirements.
